Subdirectorio del sitio web
Este campo determina qué directorio está protegido con contraseña. Puede seleccionar Todo el sitio web para controlar el acceso a todas las páginas web en el dominio, o elegir Sub-directorio e ingresar una ruta relativa al directorio public_html como estadísticas o documentos / privado .

Para proteger algunos de los scripts CGI de un dominio, seleccione el subdirectorio CGI e ingrese un subdirectorio en cgi-bin , como admin .
