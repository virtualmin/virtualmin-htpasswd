Thư mục con trang web
Trường này xác định thư mục nào được bảo vệ bằng mật khẩu. Bạn có thể chọn toàn bộ trang web để kiểm soát truy cập vào tất cả các trang web trong tên miền, hoặc chọn Sub-thư mục và nhập đường dẫn tương đối so với public_html thư mục như số liệu thống kê hoặc docs / tin.

Để bảo vệ một số tập lệnh CGI của tên miền, chọn thư mục con CGI và nhập thư mục con bên dưới cgi-bin , như quản trị viên .
