Subdirecció de llocs web
Aquest camp determina quin directori està protegit amb contrasenya. Podeu seleccionar Tot el lloc web per controlar l’accés a totes les pàgines web del domini o triar Subdirector i introduir una ruta relativa al directori public_html com ara estadístiques o docs / private .

Per protegir alguns scripts CGI d'un domini, seleccioneu subdirecció CGI i introduïu un subdirector a cgi-bin , com l' administrador .
