Sottodirectory del sito Web
Questo campo determina quale directory è protetta da password. È possibile selezionare Tutto il sito Web per controllare l'accesso a tutte le pagine Web nel dominio oppure scegliere la sottodirectory e immettere un percorso relativo alla directory public_html come stats o docs / private .

Per proteggere alcuni degli script CGI di un dominio, selezionare la sottodirectory CGI e inserire una sottodirectory in cgi-bin , come admin .
