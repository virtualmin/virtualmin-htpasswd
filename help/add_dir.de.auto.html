Website-Unterverzeichnis
Dieses Feld bestimmt, welches Verzeichnis kennwortgeschützt ist. Sie können die gesamte Website auswählen, um den Zugriff auf alle Webseiten in der Domäne zu steuern, oder das Unterverzeichnis auswählen und einen Pfad relativ zum public_html wie stats oder docs / private eingeben.

Um einige der CGI-Skripte einer Domain zu schützen, wählen Sie das CGI-Unterverzeichnis aus und geben Sie unter cgi-bin ein Unterverzeichnis ein, z. B. admin .
